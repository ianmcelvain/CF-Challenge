## Supported node/npm
- Node: 14.x
- NPM: 6.x

## Usage
```
npm install
npm run serve
```

## Libraries

- boostrap-vue (responsive grid)
- fontawesome-vue (svg icons)
- vue-js-modal (employee details modal)
- uuid (random key generation)
